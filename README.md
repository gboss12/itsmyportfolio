# Gamachis Portfolio

A modern, responsive portfolio website built with React and TypeScript showcasing my journey as a Computer Science student at Madda Walabu University.

## About

This portfolio represents my academic journey, skills, and projects as a 3rd-year Computer Science student. It features a clean, modern design with smooth animations and responsive layout.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Interactive Components**: Contact form with form validation
- **TypeScript**: Built with TypeScript for better code quality and maintainability
- **Component-based Architecture**: Modular React components for easy maintenance

## Sections

- **Header**: Navigation and hero section with introduction
- **About**: Personal introduction and academic stats
- **Education**: Academic background and coursework
- **Skills**: Technical skills organized by categories
- **Projects**: Showcase of completed and ongoing projects
- **Contact**: Contact form and social media links

## Technologies Used

- React 19
- TypeScript
- CSS3 with modern features (Grid, Flexbox, Animations)
- HTML5

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd gamachis-portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

### Building for Production

```bash
npm run build
```

This builds the app for production to the `build` folder.

## Customization

To customize this portfolio for your own use:

1. Update personal information in each component
2. Modify the color scheme in CSS files
3. Add your own projects in `Projects.tsx`
4. Update contact information in `Contact.tsx`
5. Replace placeholder links with your actual social media profiles
